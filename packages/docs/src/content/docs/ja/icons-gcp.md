カテゴリ別に整理されたGoogle Cloud Platformアイコン。YAMLで `provider: gcp` を使用します。

```yaml
nodes:
  - id: my-instance
    provider: gcp
    kind: compute.gce  # 以下のテーブルからkind値を使用
```
